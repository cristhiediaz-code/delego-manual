# Manual Delego Web

Bienvenido al manual de usuario de **Delego Web**, la plataforma de operaciones de campo y logística de última milla para **administradores** y **dispatchers**.

Esta guía te acompaña desde la configuración inicial de tu cuenta hasta la operación diaria: crear y asignar tareas, optimizar rutas, monitorear a tus agentes en tiempo real, analizar tu desempeño y mantener la trazabilidad de todo lo que ocurre.

## ¿Qué puedes hacer con Delego?

- **Asignar recursos** — pon cada tarea en manos del agente o vehículo adecuado.
- **Optimizar rutas** — deja que el motor calcule el mejor orden de visita.
- **Ejecutar en campo** — los agentes completan tareas y formularios desde su app.
- **Trazar todo** — cada cambio queda registrado para auditoría y reportes.

## Cómo está organizado este manual

- **Configuración** (Sección 3): referencia completa de catálogos y ajustes de tu cuenta.
- **Operación diaria** (Secciones 4–13): tareas, asignación, rutas, monitoreo, dashboard, notificaciones, reportes, recurrencias, auditoría e integraciones.
- **Soporte** (Sección 14): solución de problemas y preguntas frecuentes.
